# Glossary

Compact reference. ≈ 90 terms.

## Anatomy

- **Cortex** — outer 2–4 mm sheet of cerebral hemispheres; ~16B neurons in 6 layers.
- **Neocortex** — phylogenetically newer cortex (most of human cortex).
- **PFC (prefrontal cortex)** — frontmost cortex; goals, planning, working memory, executive control.
- **dlPFC** — dorsolateral PFC; working memory, rules.
- **vmPFC** — ventromedial PFC; value, decision.
- **ACC** — anterior cingulate cortex; conflict, error monitoring.
- **OFC** — orbitofrontal cortex; reward valuation.
- **M1** — primary motor cortex.
- **S1** — primary somatosensory cortex.
- **V1, V2, V4, MT, IT** — visual hierarchy areas.
- **A1** — primary auditory cortex.
- **Thalamus** — central relay between cortex and world; partner of every cortical area.
- **Hippocampus** — episodic memory, spatial cognition; subregions DG, CA3, CA1.
- **Entorhinal cortex (EC)** — gateway to hippocampus; contains grid cells.
- **Amygdala** — emotional valence, fear conditioning.
- **Basal ganglia (BG)** — striatum, GP, STN, SN; action selection, RL.
- **Striatum** — input nucleus of BG; medium spiny neurons.
- **VTA / SNc** — ventral tegmental area / substantia nigra pars compacta; dopaminergic.
- **Cerebellum** — forward models, fine motor, timing; Purkinje + granule + climbing fibers.
- **Locus coeruleus (LC)** — noradrenergic nucleus.
- **Raphe nuclei** — serotonergic.
- **NBM** — nucleus basalis of Meynert; cholinergic.

## Cell types

- **Pyramidal neuron** — excitatory cortical workhorse.
- **Interneuron** — inhibitory; PV, SOM, VIP subtypes are key.
- **Purkinje cell** — large cerebellar output neuron.
- **Granule cell** — small densely packed neuron in cerebellum / DG.
- **Medium spiny neuron (MSN)** — striatal projection neuron.

## Biophysics

- **Action potential / spike** — ~1 ms voltage pulse; the "1-bit" output of a neuron.
- **Membrane potential (V)** — voltage across the cell membrane (~−70 mV at rest).
- **Refractory period** — ~2 ms after spike when neuron cannot refire.
- **Soma** — cell body.
- **Dendrite** — input branch.
- **Axon** — output wire.
- **Synapse** — connection point between neurons.
- **AMPA, NMDA, GABA-A/B** — receptor families.
- **Ion channel** — voltage- or ligand-gated pore.

## Plasticity & learning

- **LTP** — long-term potentiation.
- **LTD** — long-term depression.
- **STDP** — spike-timing-dependent plasticity.
- **Hebbian rule** — "fire together, wire together."
- **Three-factor rule** — Hebbian × neuromodulator.
- **Eligibility trace** — short-term synaptic tag enabling delayed credit assignment.
- **Catastrophic forgetting** — losing old skills when learning new ones.
- **EWC** — Elastic Weight Consolidation.
- **CLS** — Complementary Learning Systems.

## Coding

- **Rate code** — info in firing rate.
- **Temporal code** — info in spike timing.
- **Population code** — info distributed across many neurons.
- **Sparse code** — few neurons active at any time.
- **Phase precession** — hippocampal place cells fire at progressively earlier theta phases.
- **Grid cell, place cell, head-direction cell, border cell** — spatial cell types.

## Computation & dynamics

- **Rate model / RNN** — continuous-rate neural network.
- **LIF (Leaky Integrate-and-Fire)** — simplest spiking neuron model.
- **Hopfield network** — energy-based attractor associative memory.
- **Attractor** — stable state of a dynamical system.
- **E/I balance** — excitatory/inhibitory balance giving irregular firing.
- **Gamma rhythm** — 30–100 Hz oscillation; local processing.
- **Theta rhythm** — 4–8 Hz; hippocampus & exploration.
- **Sharp-wave ripple (SWR)** — 150–250 Hz hippocampal event during replay.

## RL & decision

- **TD error** — temporal-difference reward prediction error.
- **MF / MB** — model-free / model-based RL.
- **SR** — successor representation.
- **Successor features** — basis-decomposed SR.
- **Actor-critic** — RL framework matching BG.

## Bayesian / inference

- **Predictive coding** — cortex passes predictions down, errors up.
- **Free energy** — variational upper bound on surprise (Friston).
- **Active inference** — action chosen to minimize expected free energy.
- **Markov blanket** — agent boundary in FEP.
- **PPC** — probabilistic population code.
- **Generative model** — joint over observations + hidden states.

## NeuroAI methods

- **RSA** — representational similarity analysis.
- **Brain-Score** — ANN→cortex prediction benchmark.
- **Linear probe / decoder** — readout testing what's linearly readable.
- **Neural manifold** — low-dim subspace of population activity.

## Tools / measurements

- **EEG** — scalp electrical, ms resolution, cm spatial.
- **fMRI** — BOLD signal, second resolution, mm spatial.
- **Single-unit / multi-unit ephys** — electrode in/near a neuron.
- **Two-photon imaging** — calcium fluorescence, ~Hz, micron.
- **Neuropixels** — high-channel-count silicon probes.
- **Optogenetics** — light-controlled channelrhodopsin/halorhodopsin perturbation.
- **Connectome** — full wiring diagram.

## Neuromodulators

- **Dopamine** — RPE, motivation, vigor.
- **Noradrenaline (norepinephrine)** — surprise, arousal.
- **Serotonin** — mood, time horizon.
- **Acetylcholine** — attention, plasticity gate.
- **Histamine** — wakefulness.

## Consciousness

- **GWT / GNW** — global workspace theory / global neuronal workspace.
- **IIT** — integrated information theory.
- **HOT** — higher-order theory.
- **NCC** — neural correlate of consciousness.
- **Phenomenal vs access consciousness** — what-it-is-like vs reportable.
